  <H2>
   Description</H2>
  <P STYLE="margin-left : 36.0pt;">
   This entry is one of the two entries that describe the live load 
   forces that are associated with a &#147;Truck.&#148; Truck loads 
   include these stationary loads in addition to the moving loads 
   specified in the &#147;Truck Definition&#148; entry form. Use the 
   same name on this entry as on the &#147;<A HREF="Truck_Definition.html">Truck
    Definition</A> &#148; entry to combine stationary loads with moving 
   loads for a single truck definition.</P>
  <P STYLE="margin-left : 36.0pt;">
   &nbsp;</P>
  <P STYLE="margin-left : 36.0pt;">
   Forces can be applied to member lines and COGO points explicitly. Use 
   this entry sheet to describe <SPAN STYLE="font-weight : bold;">either</SPAN> 
   a &#147;Load at Point&#148; or a member load applied to a member line.</P>
  <P STYLE="margin-left : 36.0pt;">
   &nbsp;</P>
  <P STYLE="margin-left : 36.0pt;">
   A full length uniform load can easily be applied by just entering the 
   a Start Value and leaving the End Value, Start Distance, and End 
   Distances blank.</P>
  <P STYLE="margin-left : 36.0pt;">
   &nbsp;</P>
  <H2>
   Parameters</H2>
  <P STYLE="text-indent : -58.5pt;margin-left : 94.5pt;">
   &nbsp;</P>
  <P STYLE="text-indent : -58.5pt;margin-left : 94.5pt;">
   <SPAN STYLE="font-weight : bold;">COGO Point&nbsp;</SPAN></P>
  <P STYLE="margin-left : 72.0pt;">
   If this entry is being use to describe a point load, then specify the 
   point where the force will be applied. Member loading does not use 
   this entry.</P>
  <P STYLE="margin-left : 36.0pt;">
   <SPAN STYLE="font-weight : bold;">Force Dir&nbsp;</SPAN></P>
  <P STYLE="margin-left : 72.0pt;">
   Forces are applied in the <SPAN STYLE="font-weight : bold;">global</SPAN> 
   coordinate system. Use one of the following six direction codes: FX, 
   FY, FZ, MX, My, or MZ. Note that unlike the member loads, the sense 
   of these loads is consistent with the directions of the global 
   coordinate system. Member loading does not use this entry.</P>
  <P STYLE="margin-left : 36.0pt;">
   <SPAN STYLE="font-weight : bold;">Force&nbsp;</SPAN></P>
  <P STYLE="margin-left : 72.0pt;">
   Enter the force in units of Kips and Feet. Member loading does not 
   use this entry.</P>
  <P STYLE="margin-left : 36.0pt;">
   <SPAN STYLE="font-weight : bold;">Mrb Line&nbsp;</SPAN></P>
  <P STYLE="margin-left : 72.0pt;">
   Specify which member line to apply this load.</P>
  <P STYLE="margin-left : 36.0pt;">
   <SPAN STYLE="font-weight : bold;">Load Code&nbsp;</SPAN></P>
  <P STYLE="margin-left : 72.0pt;">
   Loads are applied to the major local axis of the member. Unless 
   changed by the user, the Y direction is up and the Z direction is to 
   the right when looking up station. For convenience, the Y forces are 
   negated so that a positive value will apply a downward forces. Use 
   the following codes to apply point, linear, and moment forces:</P>
  <P STYLE="text-indent : -58.5pt;margin-left : 94.5pt;">
   &nbsp;&nbsp;<SPAN STYLE="font-weight : bold;">PY</SPAN>&nbsp;<SPAN STYLE="font-weight : bold;">Downward</SPAN> 
   point load.</P>
  <P STYLE="text-indent : -58.5pt;margin-left : 94.5pt;">
   &nbsp;&nbsp;<SPAN STYLE="font-weight : bold;">PZ</SPAN>&nbsp;Point 
   Load in right direction when looking up station.</P>
  <P STYLE="text-indent : -58.5pt;margin-left : 94.5pt;">
   &nbsp;&nbsp;<SPAN STYLE="font-weight : bold;">LY</SPAN>&nbsp;Linear <SPAN STYLE="font-weight : bold;">downward</SPAN> 
   load applied along a member.</P>
  <P STYLE="text-indent : -58.5pt;margin-left : 94.5pt;">
   &nbsp;&nbsp;<SPAN STYLE="font-weight : bold;">LZ</SPAN>&nbsp;Linear 
   right direction load applied along a member.</P>
  <P STYLE="text-indent : -58.3pt;margin-left : 94.3pt;">
   &nbsp;&nbsp;<SPAN STYLE="font-weight : bold;">MY</SPAN>&nbsp;Moment 
   about the local Y axis. Looking down on the bridge, this load will be 
   in the counterclockwise direction.</P>
  <P STYLE="text-indent : -58.5pt;margin-left : 94.5pt;">
   &nbsp;&nbsp;<SPAN STYLE="font-weight : bold;">MZ</SPAN>&nbsp;Moment 
   about the local Z axis.</P>
  <P STYLE="text-indent : -58.5pt;margin-left : 94.5pt;">
   &nbsp;</P>
  <P STYLE="text-indent : -58.5pt;margin-left : 94.5pt;">
   <SPAN STYLE="font-weight : bold;">Start Dist&nbsp;</SPAN></P>
  <P STYLE="margin-left : 72.0pt;">
   Distance from start of member line. Use to specify location of point 
   or moment forces, and beginning of linear load.</P>
  <P STYLE="margin-left : 36.0pt;">
   <SPAN STYLE="font-weight : bold;">End Dist&nbsp;</SPAN></P>
  <P STYLE="margin-left : 72.0pt;">
   Distance from start of member line. Use to specify ending location of 
   linear load.</P>
  <P STYLE="margin-left : 36.0pt;">
   <SPAN STYLE="font-weight : bold;">Start Value&nbsp;</SPAN></P>
  <P STYLE="margin-left : 72.0pt;">
   Point or moment force value, or starting linear load.</P>
  <P STYLE="margin-left : 36.0pt;">
   <SPAN STYLE="font-weight : bold;">Ending Value&nbsp;</SPAN></P>
  <P STYLE="margin-left : 72.0pt;">
   Ending force value for linear load.</P>
  <P STYLE="margin-left : 36.0pt;">
   <SPAN STYLE="font-weight : bold;">Description&nbsp;</SPAN></P>
  <P STYLE="margin-left : 72.0pt;">
   Optional description.
