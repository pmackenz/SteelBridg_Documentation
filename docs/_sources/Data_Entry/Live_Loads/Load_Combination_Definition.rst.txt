<H2>Description</H2>
<P style="margin-left:36.00pt;">This table is used to reference sets of trucks, lanes, and factors into a set that are combined for capacity and rating calculations. </P>
<H2>Parameters</H2>
<P style="margin-left:72.00pt;text-indent:-36.00pt;"><span style="font-weight:bold; ">Name</span></P>
<P style="margin-left:72.00pt;">Specify a unique name for a loading configuration.</P>
<P>&nbsp;<span style="font-weight:bold; ">Truck Group</span></P>
<P>&nbsp;&nbsp;Reference a <A href="Truck_Group_Definition.html">Truck Group</A> </P>
<P>&nbsp;<span style="font-weight:bold; ">Lane Group</span></P>
<P>&nbsp;&nbsp;Reference a <A href="Lane_Group_Definition.html">Lane Group</A> </P>
<P>&nbsp;<span style="font-weight:bold; ">Lane Load Reduction</span></P>
<P>&nbsp;&nbsp;Reference a set of lane <A href="..\Rating_and_Factors\Lane_Load_Reduction_Definition.html">load reduction factors</A> </P>
<P>&nbsp;<span style="font-weight:bold; ">Rate Location</span></P>
<P>&nbsp;&nbsp;Reference a <A href="..\Rating_and_Factors\Rate_Location.html">rate location</A> collection name.</P>
<P>&nbsp;<span style="font-weight:bold; ">Std</span></P>
<P>&nbsp;&nbsp;Put a <span style="font-weight:bold; ">Y</span> in this column to do a “Standard / WSDOT” capacity and rating calculation.</P>
<P>&nbsp;<span style="font-weight:bold; ">Inv</span></P>
<P>&nbsp;&nbsp;Put a <span style="font-weight:bold; ">Y</span> in this column to do an Inventory capacity and rating calculation.</P>
<P>&nbsp;<span style="font-weight:bold; ">Opr</span></P>
<P>&nbsp;&nbsp;Put a <span style="font-weight:bold; ">Y</span> in this column to do an Operating capacity and rating calculation.</P>
<P>&nbsp;<span style="font-weight:bold; ">OverLoad</span></P>
<P>&nbsp;&nbsp;Put a <span style="font-weight:bold; ">Y</span> in this column then the Overload <A href="..\Rating_and_Factors\Rating_Factors.html">ultimate factor</A> for denominator live loads will be used.</P>
