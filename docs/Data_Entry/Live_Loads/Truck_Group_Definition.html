In the<A href=”Load_Combination_Definition.html”> Load Combination Definition</A> both a Lane Group name and a Truck Group name are referenced. Refer to the <A href=”....Load_GenerationLoad_Generation.html”>Live Load Theory</A> section of this manual to understand how these entries are used.</P> <P style=”margin-left:36.00pt;”>&nbsp;</P> <P style=”margin-left:36.00pt;”>If there are more lanes then trucks, then the last truck in the group is repeated. Thus it is very important for overload groups, because the overload truck must be listed <span style=”font-weight:bold; “>first</span> so that it is not repeated.</P> <P style=”margin-top:0;margin-bottom:0”>&nbsp;</P> <P style=”margin-left:36.00pt;”>Note that the order that the trucks are listed in this table is the order that the trucks will be placed in the corresponding lanes that are listed in the associated Lane Group. For a bridge that is transversely symmetric, this is probably not a concern, but it is up to the engineer to specify enough different truck groupings to produce the maximum effects. This specifically refers to where in a group of trucks the heavier trucks are placed. Repeat this name for each additional lane in the group.</P> <P style=”margin-left:72.00pt;text-indent:-36.00pt;”><span style=”font-weight:bold; “>Truck Name&nbsp;</span></P> <P style=”margin-left:108.00pt;text-indent:-36.00pt;”>Reference a ?Truck? that specified in either the <A href=”Live_Load_Definition.html”>Live Load Definition</A> , <A href=”Truck_Definition.html”>Truck Definition</A> , or <A href=”Live_Load_Envelope_Definition.html”>Live Load Envelope definition</A> . </P> <P style=”margin-left:72.00pt;text-indent:-36.00pt;”><span style=”font-weight:bold; “>Axle Force Multiplier</span></P> <P style=”margin-left:72.00pt;text-indent:-36.00pt;”><span style=”font-weight:bold; “>&nbsp;</span>The values from the envelope data for the referenced ?truck? will be multiplied by this factor.</P> <P style=”margin-left:72.00pt;text-indent:-36.00pt;”><span style=”font-weight:bold; “>Numerator Denominator Code</span></P> <P style=”margin-left:72.00pt;text-indent:-36.00pt;”>&nbsp;All live load envelopes have two parts, a numerator and a denominator part. This is required because some forms of the rating equation have live load data in both the numerator and denominator. Specify in this field which envelope this truck?s forces are to be applied. Use a ?N? for numerator, and a ?D? for denominator. </P> </BODY> </HTML>
